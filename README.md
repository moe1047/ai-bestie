## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your environment variables:
   ```bash
   GROQ_API_KEY=your_api_key_here
   ```

## Usage

Run the main script to test Vee's conversation capabilities:
```bash
python3 main.py
```
